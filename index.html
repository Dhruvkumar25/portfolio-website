Projects

ML-Based COVID-19 Detection from Lung CT Images

Developed a machine learning framework using Tensor-Empirical Wavelet Transform (Tensor-EWT) and Gabor features for COVID-19 diagnosis from lung CT images. Feature selection was performed using Neighborhood Component Analysis (NCA) and Student’s t-value, followed by classification using LS-SVM.

Achieved 96.29% accuracy, 96.56% specificity, 96.03% sensitivity, and 95% F1-score. This work has been published in IEEE.

Crop Disease Detection Using InceptionV3 Transfer Learning

Built a deep learning-based crop disease detection system trained on over 54,000 PlantVillage images covering 14 different crops. Utilized InceptionV3 transfer learning to achieve high generalization.

Achieved 91.97% validation accuracy. Integrated TensorFlow Lite for on-device Android inference and combined the system with IoT sensor monitoring. Submitted to IEEE.

Hybrid CNN–XGBoost Model for Animal Respiratory Disease Detection

Developed a hybrid deep learning framework combining CNN-based feature extraction with XGBoost classification to detect respiratory diseases in livestock using thermographic and RGB images.

Achieved over 90% precision and recall for early, non-invasive disease detection, outperforming traditional methods and enabling real-time health monitoring in farm environments.

Kidney Ultrasound Classification via PCA-Guided Multi-Domain Feature Fusion

Proposed a multi-domain feature fusion approach combining deep learning and wavelet-based handcrafted features. Applied PCA-guided feature selection to enhance classification performance.

Achieved state-of-the-art accuracy and robustness for multi-class kidney disease classification.

Medical Imaging Dataset Creation – NTU

Collaborating on a large-scale kidney medical imaging dataset involving dataset structuring, annotation planning, preprocessing (CLAHE, resizing, normalization), and metadata management for future AI-driven disease detection research.
